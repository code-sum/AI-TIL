### [CVPR 2025] The Point, the Vision and the Text: Does Point Cloud Boost Spatial Reasoning of Large Language Models? [🗂️](https://arxiv.org/abs/2504.04540)

- 3줄 요약
  - 3D LLM 이 공간 추론을 할 때 point cloud 데이터가 실제로 얼마나 기여하는지 분석한 결과, 예상외로 그 역할이 절대적이지 않다는 사실을 발견.
  - 실험 결과, 모델들은 point cloud 없이 시각이나 텍스트 정보만으로도 충분히 경쟁력 있는 성능을 냈고, 오히려 복잡한 공간 관계나 세밀한 좌표 데이터를 활용하는 데는 여전히 한계를 보임.
  - 본 연구는 현재 3D 모델의 구조적 약점을 지적하며, 향후 더 정교한 공간 지각 능력을 갖춘 차세대 모델로 발전하기 위한 통찰을 제공.

- Introduction
  - 3D LLMs (Point Cloud + LLM)
    - Structure: Point Cloud Data > Encoding {Feature Extraction} > Embedding (3D to LLM) > Token > LLM
    - Problem Statement
      - "Do the 3D LLMs Really Utilize the Point Cloud Information?"
      - "Do 3D LLMs Perform the Spatial Reasoning?"
    - Abstract
      - Multi-modality Evaluation: Ask the 'Unlearned Pattern', which can be answered with spatial reasoning

- Methodology
  - Benchmark Dataset: ScanReQA
  - RelSpatial QA (Relationship)
    - Inversing the Spatial Relationship
      - By LLM-assisted Module
      - Human Annotation (Address Hallucination)
