### [CVPR 2025] MBQ: Modality-Balanced Quantization for Large Vision-Language Models [🗂️](https://arxiv.org/abs/2412.19509)

- 3줄 요약
  - 시각-언어 모델(VLM)은 다양한 실세계 애플리케이션을 가능하게 했지만, 거대한 파라미터 크기로 인한 대규모 메모리 및 연산 오버헤드는 배포에 큰 걸림돌이 되고 있음
  - 이를 해결하기 위해 포스트 트레이닝 양자화(PTQ) 적용되고 있지만, 기존 방식들은 모달리티 간의 차이를 고려하지 않고 대규모 언어 모델(LLM)에만 초점을 맞추어 정확도 손실 발생
  - 본 논문에서는 시각과 언어 토큰 간의 뚜렷한 민감도 차이를 발견하고 이를 보정 과정에 반영한 '모달리티 균형 양자화(MBQ)' 방식을 제안하며, 이는 기존 SOTA 베이스라인 대비 최대 11.6%의 정확도 향상과 RTX 4090 기준 1.4배의 속도 향상을 달성

- Backgrounds
  - Quantization
    - Convert high-precision floating-point numbers(FP32) to lower-precision formats (e.g. INT4, FP4, ≤ 3bit)
    - Apply on weights and activations
    - Model size, memory overhead, computational cost decreases

- Motivation: "Differences in modality sensitivity must be taken into account!"
  - Existing LLM-centric PTQs treat language tokens and vision tokens equally
  - Data: visual data contains a high degree of redundancy > More tolerant for small perturbations
  - Model: generated content is primarily biased by LLMs rather than input images
  - Toy example with COCO dataset (image-caption pair)
    - Gradients of loss fuction (∂L/∂Y)
    - Language tokens impact the loss fuction more than 10x as much as vision tokens

- Methods
  - Modality-Balanced Quantization (MBQ)
    - Proof the relationship between gradient and sensitivity
      - $L(\mathbf{Y} + \Delta) \simeq L(\mathbf{Y}) + \mathbf{g}^T * \Delta$
    - Search optimal equalization factors (E) considering MBQ (Calibration stage)
