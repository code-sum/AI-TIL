### [CVPR 2025] Flowing from Words to Pixels: A Noise-Free Framework for Cross-Modality Evolution [🗂️](https://arxiv.org/abs/2412.15213)


- Diffusion & Flow Matching
  - Diffusion
    - Gaussian noise > target data 분포로의 매핑 학습
    - Cross-modal (예: Text-to-Image) 에서는 텍스트가 조건(condition)으로 추가됨 (주로 cross-attention 활용)
  - Flow Matching
    - Diffusion 의 일반화 버전 (더 넓은 개념)
    - Noise 에서 target 으로 가는 더 단순하고 최적화된 경로 학습
    - Source 분포가 반드시 noise 일 필요는 없음 > target 과 상관관계 있는 다른 분포 (예: text latent) 사용 가능

- Background
  - RQ: Flow Matching 방식을 사용해 Noise, Condition 없이 직접 modality 간 매핑이 가능할까?
  - (1) Shape Mismatch
    - Diffusion/Flow 모델은 source 와 target 분포의 shape 이 동일해야 함
    - ➡️ 서로 다른 모달리티(예: 텍스트 vs 이미지)는 구조가 달라서 직접 매핑 불가
  - (2) Classifier-Free Guidance(CFG) 적용 불가
    - 기존 SOTA 모델들은 CFG 로 성능 향상
    - Flow Matching 방식 채택 시, 조건 정보가 source data 자체에 포함되어 있음
    - ➡️ 기존 CFG 방식(조건 on/off 전환) 사용 불가능
