### [ICASSP 2023] Large-scale Contrastive Language-Audio Pretraining with Feature Fusion and Keyword-to-Caption Augmentation [🗂️](https://arxiv.org/abs/2211.06687)

- 3줄 요약
  - 대규모 Audio-Text 쌍(633,526개)으로 구성된 새로운 데이터셋 LAION-Audio-630K를 구축하여 공개
  - 가변 길이의 오디오를 처리하고 성능을 높이기 위해 특징 융합 및 키워드-캡션 증강 기법을 도입한 Contrastive Language-Audio Pretraining (CLAP) 모델을 제안
  - 실험 결과, 제안된 모델이 텍스트-오디오 검색 및 zero-shot 오디오 분류 작업에서 최고 수준(SOTA) 성능을 달성

- Model Architecture
  - CLAP
    - Audio Encoder
      - (1) PANN (2048-d)
      - (2) HTSAT (768-d)
    - Text Encoder
      - (1) CLIP Transformer (512-d)
      - (2) BERT (768-d)
      - (3) RoBERTa (768-d)
    - MLP layer: mapping
      - Loss Function
        - $$L = \frac{1}{2N} \sum_{i=1}^{N} \left( \log \frac{\exp(E_i^a \cdot E_i^t / \tau)}{\sum_{j=1}^{N} \exp(E_i^a \cdot E_j^t / \tau)} + \log \frac{\exp(E_i^t \cdot E_i^a / \tau)}{\sum_{j=1}^{N} \exp(E_i^t \cdot E_j^a / \tau)} \right)$$
  - Feature Fusion
    - Attentional Feature Fusion
      - T < 10-second: repeat + pad
      - T > 10-second:
        - (1) Extract local features & global feature
        - (2) Send features into Conv2D
        - (3) Send local latent features into Merge Conv2D
        - (4) Attentional Feature Fusion
          - $$$$
