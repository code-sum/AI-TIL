### [ACL 2025] Meta-rater: A Multi-dimensional Data Selection Method for Pre-training Language Models [🗂️](https://aclanthology.org/2025.acl-long.533/)

- 3줄 요약
  - dd

- 연구배경 & 문제제기
  - 데이터 품질의 중요성
    - LLM 의 핵심 역량은 사전학습 과정에서 형성되며, 데이터 품질이 모델 성능의 결정적 요인
    - 오픈소스 및 독점 LLM 의 사전학습 데이터 구성은 대부분 비공개
  - 기존 방법의 한계
    - 단일 차원 평가에 집중 (피상적 텍스트 특성)
    - 중복성 감소를 우선시, 내재적 품질 평가 부족
  - Meta-rater 는 전문성, 가독성, 추론, 청결성이라는 4가지 새로운 평가 차원을 통해 전체론적 데이터 품질 평가를 수행

- Method: 
  - PRRC 프레임워크 제작 
    - (1) Natural Language Quality Signals: RedPajama (문장/단어수, 알파벳 비율 등 rule-base 기반 자연어 평가)
    - (2) Data Importance Scores: 텍스트-고품질 도메인 유사도 측정
    - (3) Model-Base Ratings: Rating 을 위한 4가지 차원별 model 제작 (ModernBERT base)
  - 4가지 핵심 평가 차원
    - 전문성 (Professionalism): 텍스트에 포함된 전문 지식의 수준을 평가 (QuRating, QA task)
    - 가독성 (Readability): 텍스트 이해 용이성 측정 (문장 길이, 단어 길이, 음절 수, 단어 빈도 등 고려)
    - 추론 (Reasoning)
      - 다단계 논리적 추론, 심층 분석, 정보 통합을 요구하는 정도를 평가
      - 높은 추론 복잡성 + 소규모 LLM 의 추론 능력 향상
    - 청결성 (Cleanliness)
      - 올바른 형식 작성 여부 (부적절한 문자, 하이퍼링크, 광고 등 노이즈 부재 확인)
      - 모델 기반 접근법, 불규칙 현상이나 이상 현상(long tail) 처리에서 우수한 일반화 능력
  - PRRC 평가 모델 훈련
    - Annotation: Llama-3.3-70B-Instruct 모델을 활용하여 SlimPajama 에서 샘플링한 50만개 예시에 주석 달기
    - ModernBERT 를 각 차원별 평가 모델로 파인튜닝
  - 성능: 87-92% 의 F1 Score 달성

- Meta-rater Optimal Weight Training Process
  - Goal 설정: 데이터에 다차원 품질 점수(PRRC 포함 25가지)를 매겨 추후 통합 & 데이터 선택을 위한 단일 통합 점수 도출
  - 프록시 모델 활용: 수백개의 소규모 프록시 모델을 훈련하여 품질 점수 가중치와 검증 손실 데이터를 생성
  - Regression: 수집된 데이터 포인트 {(wi, li)}를 사용하여 LightGBM 기반 회귀 모델을 학습하고, 품질 점수 가중치에 따른 검증 손실을 예측
  - 최적 가중치 예측: 학습된 회귀 모델을 통해 더 넓은 가중치 공간을 탐색하고, 예측 손실을 최소화하는 최적 가중치 w* 를 식별하여 최종 LLM 훈련용 최상위 데이터 선택

- 실험 설정 및 주요 결과
  - 실험 설정
    - 데이터셋: SlimPajama (627B 토큰) 에서 30B 샘플링
    - 모델: 1.3B, 3.3B, 7.2B 파라미터 Transformer 디코더
    - 평가 태스크: ARC, HelloSwag, RACE, MMLU 등
    - 베이스라인: Random, PPL, DSIR, QuRating 등 12개 방법
