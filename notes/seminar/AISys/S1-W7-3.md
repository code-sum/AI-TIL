### [EuroSys 2025] CacheBlend: Fast Large Language Model Serving for RAG with Cached Knowledge Fusion [🗂️](https://arxiv.org/abs/2405.16444)

- Motivation
  - prefill 단계에서 발생하는 latency 문제 > TTFT 시간 개선해야함
  - 기존 KV cache 재사용 방법은 두 청크 사이 cross-attention 누락으로 인해 LLM 품질 저하 발생

- CashBlend
  - CacheBlend 는 pre-computed KV cache 를 재사용하면서도, High-KV-Deviation 토큰을 선택적으로 recompute 하여 cross-attention 을 복구하고, 이 과정을 KV cache 로딩과 파이프라인하여 지연 없이 효율성을 극대화
    - 선택적으로 recompute (정답과 편차가 심한 부분만 재연산)
