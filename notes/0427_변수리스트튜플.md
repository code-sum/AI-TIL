휴보 코드 실습

- 객체는 이름 여러개를 가질 수 있음
- 이름은 여러 객체를 가리킬 순 없음

```python
from cs1robots import *
import time

create_world(avenues = 5, streets = 5)
sleep_time = 0.5

hubo = Robot("yellow")
time.sleep(sleep_time)
hubo.move()
ami = hubo  # <- ami, hubo 는 동일한 객체
time.sleep(sleep_time)
ami.turn_left()
time.sleep(sleep_time)
hubo.move()

time.sleep(sleep_time)
hubo = Robot("blue")  # 
time.sleep(sleep_time)
hubo.move()
time.sleep(sleep_time)
ami.turn_left()
time.sleep(sleep_time)
ami.move()
```
